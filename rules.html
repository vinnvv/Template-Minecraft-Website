SERVER GUIDELINES

Peraturan Server

1. No Cheating

Dilarang menggunakan client modifikasi (X-Ray, Fly, KillAura). Hukuman: Permanent Ban.

2. Respect Others

Hormati sesama pemain dan staff. Dilarang rasisme, toxicity, atau pelecehan dalam bentuk apapun.

3. No Griefing

Dilarang merusak bangunan orang lain tanpa izin. Kami memiliki sistem log untuk melacak pelaku.

4. No Spam

Dilarang spam di chat, baik berupa pesan berulang, iklan server lain, maupun karakter tidak jelas.

5. Patuhi Staff

Keputusan staff bersifat final. Jika merasa tidak adil, ajukan banding lewat Discord resmi kami.

6. Laporkan Bug

Jika menemukan bug, segera laporkan ke staff. Mengeksploitasi bug untuk keuntungan pribadi dilarang.
